# QHO444 AE2 – Disneyland Reviews Analysis

This project is part of the QHO444 Problem Solving Through Programming module.

The program analyses Disneyland review data and provides text-based
summaries and visualisations via a menu-driven interface.

## How to run
1. Ensure Python 3 is installed
2. Open the project folder
3. Run the program:
   python main.py

## Project structure
- main.py     - Program entry point and menu control
- tui.py      – Text-based user interface and input handling
- process.py  – Data processing and analysis logic
- visual.py   – Data visualisation (charts)
- Disneyland_reviews.csv – Review dataset
